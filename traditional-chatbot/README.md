# Traditional Rule-Based Chatbot

## Overview

This project demonstrates a traditional chatbot implemented in Python using a rule-based approach without Large Language Models (LLMs).

## Features

- Greeting users
- Help command
- About command
- Date
- Time
- Weather placeholder
- Capabilities
- Invalid input handling
- Exit command
